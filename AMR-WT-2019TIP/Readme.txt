The code requires the version of Matlab2018.

Please download some related codes on gPb, OEF, and SE from the following links.

(1) gPb: https://www2.eecs.berkeley.edu/Research/Projects/CS/vision/grouping/resources.html
Reference: Arbelaez P, Maire M, Fowlkes C, et al. Contour detection and hierarchical image segmentation. IEEE Transactions on Pattern Analysis and Machine Intelligence, 2011, 33(5): 898-916.

(2) OEF: https://github.com/samhallman/oef
Reference: Hallman S, Fowlkes C C. Oriented edge forests for boundary detection//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. 2015: 1732-1740.

(3) SE: https://github.com/pdollar/edges
Reference: Doll¨¢r P, Zitnick C L. Fast edge detection using structured forests. IEEE Transactions on Pattern Analysis and Machine Intelligence, 2015, 37(8): 1558-1570.

We only present two examples:

(1) main_Sobel_AMR_WT  obtains image segmentation results using AMR-WT, where the gradient image 
generated by the Sobel operator.

(2) main_SE_AMR_WT obtains image segmentation results using AMR-SE, where the gradient image 
generated by SE.
